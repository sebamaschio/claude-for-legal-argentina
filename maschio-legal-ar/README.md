# maschio-legal-ar

Plugin de práctica forense argentina para Claude. Desarrollado para el estudio Maschio, Junín, Provincia de Buenos Aires.

## Skills incluidos

| Skill | Uso |
|---|---|
| `brief` | Briefing forense diario, research, urgentes |
| `triage-documento` | Clasificación ROJO/AMARILLO/VERDE de documentos recibidos |
| `consulta-expediente` | Estado de expediente cruzando Lex Doctor, PJN y SCBA |
| `comunicacion-fehaciente` | Cartas documento, intimaciones, respuesta a CD, oficios AFIP |
| `presentacion-electronica` | Checklist de presentación PJN / SCBA y firma de contratos |
| `briefing-audiencia` | Prep de audiencias: 360, vista de causa, SECLO, mediación, penal, cliente |
| `revision-contrato` | Revisión bajo derecho argentino con marcado VERDE/AMARILLO/ROJO |
| `riesgo-procesal` | Evaluación de viabilidad, exposición, plazos y costo de litigar |
| `cumplimiento-normativo` | Control regulatorio bajo derecho argentino |

## Fuero y jurisdicción

Fuero PBA (CPCC PBA, acordadas SCBA) y Justicia Nacional. Plazos calculados según regla martes/viernes SCBA.
